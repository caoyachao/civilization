This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a documentation repository containing a single worldbuilding document (`req.md`) that defines a 10-level Kardashev-inspired civilization classification system. The document covers:

- **Civilization tiers** (Level 1–10): progression from planetary-bound societies to transcendent, reality-defining civilizations, keyed to energy mastery and spatial scale.
- **Warfare by tier**: tactics, weapons, and strategic logic for inter-civilization conflict at each level.
- **Planetary / stellar destruction capabilities**: which tiers can destroy planets or stars and by what means.

There is no build system, test suite, or application code in this repository. The primary file is `req.md`.
